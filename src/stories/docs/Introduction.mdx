Icones

Aqui é usado os icones svg do figma e o pacote de fontes Material Symbols

Para usar um icone material é só usar o component q-icon

por exemplo para o icone de close voce utilizaria dessa forma no q-icon name='close'


Ja para icones do figma, voce ira copiar o svg do icone

transformar em symbol com um id dentro /publicicons.svg

e vai referenciar no q-icon da seguinte forma

name="svguse:/icons.svg#icon-1"
